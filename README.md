# Gill M Edwards - Website

Website for Gill M Edwards, Stress Disruptor & Reiki Practitioner.

## Pages

- **Home** (`index.html`) - Welcome, services overview, book, singalong preview
- **About** (`about.html`) - Gill's story, journey, and approach
- **Services** (`services.html`) - 1:1 Stress Coaching, Reiki Healing, Life Coaching
- **Singalong** (`singalong.html`) - Community singing group in Stoke-on-Trent
- **Contact** (`contact.html`) - Contact form and social media links

Plain HTML, CSS, and JavaScript. No build tools required.

Design inspired by the "Stress Less? Hell Yes!" book cover - soft blush pink, bold red, and clean white.
